# 🛰️ GeoTIFF Object Detector — Инструкция по запуску

## Что нужно заранее
- Windows 10/11
- Python 3.9–3.12 (скачать с [python.org](https://www.python.org/downloads/))
- Файл модели `best.pt` (уже лежит в папке)

---

## Шаг 1 — Установить зависимости

Открой папку `CDApp`, зажми **Shift** и **правой кнопкой мыши** кликни на пустое место → выбери **«Открыть окно PowerShell здесь»**.

В открывшемся окне вставь и нажми Enter:

```
pip install -r requirements.txt
```

Дождись завершения (может занять 2–5 минут).

---

## Шаг 2 — Запустить приложение

В том же окне PowerShell вставь и нажми Enter:

```
python -m streamlit run app.py
```

---

## Шаг 3 — Открыть в браузере

Браузер откроется автоматически. Если нет — вручную перейди по адресу:

```
http://localhost:8501
```

---

## Как пользоваться

1. В левой панели нажми **«Browse files»** и выбери свой GeoTIFF снимок
2. Настрой **«Порог уверенности»** (по умолчанию 0.25 — подходит для большинства случаев)
3. Нажми кнопку **«🚀 Запустить обработку»**
4. Дождись завершения (прогресс-бар на экране)
5. Смотри результаты на карте → скачивай **GeoJSON** или **GeoPackage**

---

## Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| `python` не найден | Переустанови Python с галочкой **«Add to PATH»** |
| `streamlit` не найден | Выполни `pip install streamlit` |
| Ошибка при загрузке файла >200 МБ | Файл `.streamlit/config.toml` уже настроен на 1 ГБ |
| Приложение не открывается | Проверь, не занят ли порт 8501 другой программой |
